# Lite_Version_Calendar
Удалена кнопка сохранения в PDF и весь связанный с ней код
Исправлены стили для печати в сохраненном HTML:
Добавлено transform: scale(1.4) для увеличения календаря до размера A3
Сохранены оригинальные размеры ячеек (150px) и расстояния (gap: 12px)
Добавлен @page стиль для указания формата A3 landscape
Установлены минимальные поля (10mm)
Улучшено масштабирование:
Календарь центрируется и масштабируется для заполнения страницы A3
Сохранены все оригинальные пропорции и расстояния
Теперь при сохранении HTML и последующей печати в PDF:
Календарь будет занимать всю страницу A3 в ландшафтной ориентации
Расстояния между картинками будут такими же, как в оригинальном HTML
Размеры ячеек и изображений сохранят свои пропорции
Поля будут минимальными (10mm)
Для получения PDF пользователю нужно:
Сохранить HTML файл
Открыть его в браузере
Нажать Ctrl+P (Печать)
Выбрать "Сохранить как PDF"
Убедиться, что в настройках печати выбран формат A3 и ландшафтная ориентация
